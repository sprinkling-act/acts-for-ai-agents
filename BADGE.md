# Display the ACTS-aware badge

If your agent uses the ACTS framework prompts, you MAY display this badge in your README.

---

## What this badge means

This badge indicates that the developer has **read the 4 ACTS framework** and **integrated at least one** of the system prompts in this repository.

> **This is not a compliance certification.**
> **This is not a Sprinkling Act audit.**
> **This is not a verification by any third party.**

It is a developer self-declaration that they have considered the four ACTS axes — Accountability, Consent, Traceability, Skill — when building their AI agent.

For an actual assessment of your agent's EU AI Act position, run the free diagnostic at **https://sprinklingact.com/check** (9 questions, 60 seconds, no account).

---

## Markdown to copy-paste

```markdown
[![ACTS-aware](https://img.shields.io/badge/ACTS--aware-4_axes_considered-1a3cba)](https://sprinklingact.com/ai-agents)
```

This produces:

[![ACTS-aware](https://img.shields.io/badge/ACTS--aware-4_axes_considered-1a3cba)](https://sprinklingact.com/ai-agents)

The image is hosted by [shields.io](https://shields.io), not Sprinkling Act. If shields.io goes down, the badge breaks. There is no central registry that depends on it.

---

## Misuse

Displaying this badge while ignoring the framework is a misrepresentation. **Don't.**

If you see misuse, report it at https://sprinklingact.com/report-issue.
